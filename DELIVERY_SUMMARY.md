## 🔄 How It Works

1. **User Opens App** → `index.html` loads
2. **Scripts Load** → `js/api.js`, `js/auth.js`, `js/router.js`, `js/app.js`
3. **Styles Load** → `css/style.css`, `css/responsive.css`
4. **App Initializes** → Router checks authentication
5. **Home Page Shows** → User can login/register
6. **User Logs In** → JWT token stored, dashboard shown
7. **User Navigates** → Router renders appropriate page
8. **API Calls** → Frontend communicates with backend
9. **Data Updates** → Pages display data from backend
10. **User Logs Out** → Session cleared, home page shown
